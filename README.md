# AloneInTheDarkReHaunted
A heavily enhanced port of Alone In The Dark 1, with HD-4K background replacements, QOL improvements, controller support and so much more.

-Full Game Controller Support with analog control.
-Completely Re-drawn HD-4K Backgrounds that fit much better with the 3d models and keeping the original style and mood
-Shadows added to actors
-For the first time, the full CD-quality OST on PC
-Main Menus updated
-Added shadows
-Huge amount of QOL features, original game bug fixes and so much more.
